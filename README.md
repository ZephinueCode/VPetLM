# VPetLM: 一个语言模型赋能的虚拟桌面宠物

目前实现了一个最小可工作原型，支持桌面宠物的一般行为，以及可配置的 LLM 管理的对话、情绪、好感度以及关系行为。

通过 JSON 文件可以保存桌宠的长期记忆和中期记忆。

甚至还支持一个编程模式。（调用编程模型“相对严肃地”写代码）

不过目前有一个问题就是 Token 消耗实在是太高了。

欢迎和我一起开发。

## 使用方法

### 下载文件

如果你拥有 ``git``：

``git clone https://github.com/ZephinueCode/VPetLM``

如果你不会使用 ``git``，可以右上角下载 ``zip`` 文件，解压缩即可。

### 配置环境

请确保你的电脑拥有合适的 ``Python`` 环境：``python >= 3.10``。如果你没有，可以去 ``python.org`` 下载。

游戏在 ``WSL`` 上的 ``python 3.12`` 上开发，理论上来说可以在任意系统的 ``python >= 3.10`` 环境下运行。

之后，需要安装依赖。使用控制台访问这个项目的根目录（``VPetLM``），然后运行：``pip install -r requirements.txt``。

### 运行游戏

只需要安装环境后在根目录的一行代码：``python -m src`` 即可运行。运行游戏后，首先需要进行 ``API`` 和网络的配置。

如果你不知道什么是 ``API_KEY``，那我建议你等之后的稳定版发布后再尝试游玩。否则，按照基本原理配置 ``API_KEY``、``BASE_URL`` 和你想使用的模型即可。
